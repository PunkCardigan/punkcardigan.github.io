This assignment was to practice using and revising search tactics "to improve search results and learn to manage results and search statements effectively using library databases". For my topic, I chose to answer the question "What materials were used by the Incas to produce textiles before Spanish colonization?"

[Search Tactics & Results Management (Application)(PDF)](/files/searchtactics.pdf)

From this assignment I feel that I have accomplished element 6A and 6B of the Reference and User Services Competencies most fully.

- 6A. Employ techniques used to discover, retrieve, evaluate, and synthesize information from diverse sources for use by varying user populations and information environments.
- 6B. Understand and apply methods and practices necessary to provide consultation, mediation, instruction, and guidance in using recorded knowledge and information for all user populations and information environments. Emphasize problem-solving skills to determine informational needs during the reference interview process.

There were a few tools that I didn’t have much experience with, such as research and reference databases, since I didn’t learn about them or how to use them in my undergraduate education in a way that was meaningful. All of the skills like using database indexes and the multitude of search options, like proximity, are things I have only encountered since taking classes at UI, and the Application assignments in the second half of this class were structured in an iterative way that really helped me learn how to use these skills. By the final assignment, I wasn’t confused about why I wasn’t getting the results I wanted, I was just annoyed, because I knew that what I was doing should be working.

The kinds of search strategies that the textbook mentioned were fascinating to me. This is another area where I have practical experience but appreciate the more theoretical significance of what I’m doing. My organic search style is a combination of building block and pearl growing, with a little bit of shot in the dark added in. It’s useful to be able to pull apart the different methods that I use so that I can discern which ones are helping me and to choose other methods deliberately when I am struggling.

I plan to continue learning about and using different resources to perform searches and find information for patrons in ways that are meaningful to them. I’m always curious about how search engines and other querying tools work behind the scenes and I want to learn more about innovative ways to reduce barriers to accessing information for everyone.
